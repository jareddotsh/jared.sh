# jared.sh

A minimal starter website for `jared.sh`.

Files added:
- `index.html` — homepage
- `styles.css` — simple styles
- `script.js` — small interactions

Run locally (from the repo root):

```bash
# Using Python 3 built-in server (works on macOS)
python3 -m http.server 8000
# Open http://localhost:8000 in your browser
```
